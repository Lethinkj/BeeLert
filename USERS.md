## Bot Features

- **Daily Reminders**: Get personalized daily reminders via DM
- **Meeting Scheduler**: Schedule team meetings with automatic reminders
- **Voice Tracking**: Automatic meeting summaries for voice channel sessions
- **AI Motivation**: Daily AI-generated motivational messages

## How to Use

1. **Set up Daily Reminder**: Send `reminder` in DM to the bot
2. **Check Status**: Send `status` in DM to see your settings
3. **Schedule Meeting**: Use the Meeting Manager interface in the schedule-meet channel
4. **View Meetings**: Click "View Meetings" to see upcoming meetings
5. **Help**: Send `help` in DM for all available commands
